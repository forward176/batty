# Batty
## Описание игры
Batty — компьютерная игра, клон популярной игры Arkanoid на ZX Spectrum. Принцип игры: шарик отскакивает от платформы и ударяет по блокам, как только блоков не осталось – переходим на следующий уровень.

## Зависимости 
Для работы нужно установить python (версия не младше 3.10), а также библиотеку pygame (желательно версию не младше 2.1.2 )

## Инструкция для запуска игры
1. установите python версии 3.10 или старше
2. клонируйте файлы реплзитория
3. установите игру

## Инструкция для пользователя игры
- За передвижение платформы вправо отвечает rightarrow (Стрелка вправо на клавиатуре)
- За передвижение платформы влево отвечает leftarrow (Стрелка влево на клавиатуре)
- Чтобы начать уровень заново нужно нажать tab на клавиатуре
- Чтобы закончить игру нужно нажать на крестик в правом верхнем углу, тогда 'pygame window' закроется
- При разрушении всех блоков уроввень повышается. Таймер между уровнями около 2,5 секунд
